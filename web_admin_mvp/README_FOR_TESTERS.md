# 桃園中原福德宮 Web 後台預覽版

這是一份畫面預覽包，只用來測試後台畫面流程與操作是否好懂。

## 重要提醒

- 不連接正式 LINE Bot。
- 不寫入正式 Google Sheets。
- 不會正式發送 LINE / Facebook / VOOM。
- 資料皆為測試資料。
- 請勿輸入真實個資、電話、帳務資料。

## 建議開啟方式

建議使用電腦解壓縮後開啟。

不建議在手機檔案 App 直接預覽 `index.html`，因為可能只看到文字版，或畫面無法互動。

正確開啟方式：

1. 解壓縮整個資料夾。
2. 確認 `index.html`、`styles.css`、`app.js`、`mockData.js` 在同一個資料夾。
3. 用 Chrome / Edge / Safari 開啟 `index.html`。

## 請協助回饋

請以一般使用者角度測試，不需要懂程式。

回饋時可特別注意：

- 看不看得懂。
- 是否知道下一步要按哪裡。
- 哪些名稱不自然。
- 哪些按鈕會讓人不敢按。
- 長輩或廟方人員是否容易操作。

這份預覽包不是正式系統，也不是正式上線版本。
